Finding, changing, and modifying groups of elements

Loops: review

Loops allow us to call a method on every item in a group efficiently.

With jQuery, we have a method called each that we can use.

You can use each with an anonymous function, or with a named function.

Anonymous functions

each can take an anonymous function:

var links = $('a');

  links.each(function(index, elem) {
  var link = $(elem);
  link.html(link.html() + '!');
});

Named functions

Or a named function:

var processElem = function(index, elem) {
  var link = $(elem);
  link.html(link.html() + '!');
};

var links = $('a');

links.each(processElem);

Clicks, hovers, and more

Lots of events

Event documentation

There are tons of events.

Seriously, there are a ton. Don't try and memorize them.

A few of the popular events are...

Mouse events

  • Clicking: click(), actually clicking on something with your mouse
  • Double clicking: dblclick()
  • Hovering: hover(), mouse entering and then leaving an element
  • Entering and exiting: mouseenter() and mouseleave()


You can use all of these as their own functions, or with on(). Most people use them as their own functions, which is what we are going to practice.

click()

With an anonymous function:

$('a').click(function (event) {
  event.preventDefault();
  console.log('Not going there!');
});

Wait, wat.

What is event.preventDefault()?

preventDefault()

When you click something, a default behavior happens. If it's a link, that behavior is going to where the href points to.

event.preventDefault(); tells jQuery to not do the default behavior: the link doesn't take you anywhere, you stay exactly where you are on the page.

click()

click() can also be used with a named function:

function onButtonClick() {
  console.log('clicked!');
}

$('#button').click(onButtonClick);

Form events

Form specific:

  • Submitting a form: submit(), when you submit the form ("enter" key, or clicking a submit button)


Mostly used with forms, but not form specific:

  • Blurring: blur(), cursor leaves a form field or element
  • Focusing: focus(), cursor enters a form field or element; this is also used to help manage screenreaders and what they focus on after events and actions

submit()

Looks a lot like click(), doesn't it?

$('form').submit(function (event) {
  event.preventDefault();
  console.log('Not submitting, time to validate!');
});

Just like with click, we are preventing the default behavior again. The behavior we're preventing is form submission.

You can also use submit() with a named function.

Effects & Animation

Fun pretty things

Adding a little flair to our page

fadeIn()

You can make pretty things happen automatically:

$('#error').fadeIn();

Or after an event:

$('a').click(function (event) {
  event.preventDefault();
  $('#error').fadeIn();
});

Documentation

Effect options

Most effects and animations have multiple options that can be passed as arguments. You can control things like timing, delay, and speed.

It is best to consult the documentation for the method you want to use.

fadeIn() with options

These options control duration (default: 400 ms), easing (default: "swing"), and a function to do upon completion.

$("#menu").click(function() {
  $("#nav").fadeIn(200, "linear", function() {
  console.log("Faded in!");
});
});

Effects
